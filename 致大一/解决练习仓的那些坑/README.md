
### 解决  git clone 后找不到.git文件的问题

#### question 问题描述：
在执行：

> git clone https://github.com/seven-innovation-base/Git2Github-practice.git 
    
命令后，会生成一个.git的文件，但事实上你可能无法在clone到的文件位置中看到它。

#### unofficial 解释

>  可能是你的电脑隐藏文件的设置，将.git文件隐藏了。

虽然没有.git文件，但是还是可以正常操作pull和push操作。
